Place Prasanna's photo here as:  prasanna.jpg

Steps:
1. Save the photo you want displayed.
2. Rename it to:  prasanna.jpg
3. Drop it in this folder (demo-live-project/assets/).
4. Open ../birthday.html in any browser — her photo will appear
   inside the circular frame on the celebration screen.

If the file is missing, the site still works — it falls back to
a pretty "PP" monogram placeholder automatically.

Supported: .jpg (for other formats, rename the <img src> inside
birthday.html to match, e.g. assets/prasanna.png).
